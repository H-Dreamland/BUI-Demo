bui.btn

绑定页面跳转及传参

静态绑定多个按钮-一个页面只需要绑定一次

  • url跳转传参
  • 属性带参数跳转
  • disabled 属性不能跳转
  • disabled 样式不能跳转
示例:

html:

<ul id="load3" class="bui-list"> <li href="pages/ui_controls/bui.btn.html" class="bui-btn" > 不带参数跳转</li> <li href="pages/ui_controls/bui.btn.html" param='{"id":"222"}' class="bui-btn" > 带参数跳转</li> </ul>

js:

bui.btn({ id: "#page", handle: ".bui-btn" }).load();

获取页面参数

点击获取页面的参数
示例:

代码示例:

bui.getPageParams(function (res) { var param = res; bui.alert(param); });

参数说明:

callback 兼容原生的获取方式,所以都在回调里面拿.
