日期控件

1. 弹窗日期带按钮

显示年月日时分秒
显示个位不要零
显示日期
显示固定分秒

2. 没有弹窗的日期

示例:

html:

<input type="text" id="datepicker_input" readonly>

js:

// 1. 弹窗带按钮 var input = $("#datepicker_input"); var uiPickerdate = bui.pickerdate({ handle:"#datepicker_input", onChange: function(value) { input.val(value); } }); // 2. 不弹窗 var input2 = $("#datepicker_input2"); var uiPickerdate = bui.pickerdate({ id:"#datepicker_input2", onChange: function(value) { input2.val(value); } });
